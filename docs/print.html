Introduction

Everything has to start somewhere, so here we are... at the start.

This chapter will show how a program starts and how to interact with the user via the terminal/console.

Hello World

Let's start with the basics - a simple 'hello world' example. To do this we need to understand a few things. First, we need to understand where the program will start. In C and C++, this is at the main function. We will visit how this works later, but for now, all of the code that the program runs will live inside main. Create a file called main.cpp and follow the example.

main has the function signature:

int main();

This will also be explained later, but for now, let's get our hello world program working.

Our code must live inside a 'block', which is the area between two curly braces {}. This defines what is 'inside' a function. Any code that is outside of a block must be a definition or declaration (not code that runs) otherwise it will cause an error.

int main()
{
    // our code goes here
}

To be able to print a message, we need to include a file to give us access to printf. This is done using #include <stdio.h> or #include "stdio.h". The differences between these two methods will be explained later, but for now either will work perfectly fine.

#include <stdio.h>

...

Now we are ready to print our message using printf.

#include <stdio.h>

int main()
{   // the '\n' character prints a new line
    printf("Hello world\n");
}

This program will compile and run, but main expects an exit status. This is the exit status of the program once it terminates, and is used to tell if a program has failed. To do this we must 'return' a number from 0 to 255 as our exit status. 0 indicates the program was successful, anything else is an error code.

#include <stdio.h>

int main()
{
    printf("Hello world\n");

    return 0;
}

Our program is now complete - but we can't run it yet. It needs to be compiled. For a simple program like this, it is easy. Open a terminal/console in the directory containing main.cpp and type g++ main.cpp -o main to compile the program. It can now be run using ./main and you should see 'Hello world' appear in the terminal.

Note: if you have make installed, you can alternatively compile a single file project using make main.

Arguments

Sometimes we will want to supply arguments to our program when we run it. We can access these using the optional argc and argv parameters in main.

int main(int argc, char** argv)
{
    ...
}

argc is the number of arguments supplied. argv is an array of c-strings containing our arguments.

Note: the first argument (argv[0]) is the command used to run the program.

Let's create a program to add together two numbers. You can modify your main.cpp or create a new .cpp file, just make sure to compile the correct file when you are finished.

Firstly, let's make sure the user gave us enough arguments.

#include <stdio.h>

int main(int argc, char** argv)
{
    if(argc <= 2) return 1; // error exit status

    return 0;
}

It might help the user if we give them a message when they supply the wrong arguments. To do this, we can use argv[0]. It is important to mention that the if statemtnt can have it's own block. For now, all you need to know is that it can only hold one statement unless we give it a block.

#include <stdio.h>

int main(int argc, char** argv)
{
    if(argc <= 2)
    {
        printf("Usage: %s <number> <number>", argv[0]);
        return 1;
    }

    return 0;
}

The '%s' here tells printf to print a string argument, in this case argv[0]. Try compiling this and running it without any arguments.

After we know we have the correct number of arguments, we can start using them. We will assume the arguments are both numbers containing only digits for the sake of simplicity. We need another include for string conversion functions.

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char** argv)
{
    if(argc <= 2)
    {
        printf("Usage: %s <number> <number>", argv[0]);
        return 1;
    }

    float a = atof(argv[1]);
    float b = atof(argv[2]);

    printf("%f + %f = %f\n", a, b, a + b);

    return 0;
}

As with '%s', '%f' tells printf to print an argument, but in this case it is a float (aka a 'real'), which is a decimal number.

Console Input

Getting input from the console can be complicated. We need to create an empty string as well as values to store it's initial length. We then pass this to getline, which has the following signature:

ssize_t getline(char** lineptr, size_t* n, FILE* stream);

This function will return the number of characters in the string, or -1 if it fails. Let's modify our addition program to use stdin instead of command line arguments.

#include <stdio.h>
#include <stdlib.h>

int main()
{
    for(;;) // loop forever
    {
        char* in_a = NULL; // string for first argument
        size_t len_a = 0; // initial length
        printf("Enter a number: ");
        // getline call
        // 'stdin' is the file stream for console input
        if(getline(&in_a, &len_a, stdin) < 0)
        {
            // clean up the string
            free(in_a);
            // exit
            return 0;
        }

        char* in_b = NULL;
        size_t len_b = 0;
        printf("Enter a number: ");
        if(getline(&in_b, &len_b, stdin) < 0)
        {
            free(in_b);
            // exit
            return 0;
        }

        float a = atof(in_a);
        float b = atof(in_b);

        printf("%f + %f = %f\n", a, b, a + b);

        // clean up the strings
        free(in_a);
        free(in_b);
    }    
}

As you can see, this program has no end point - it loops forever. To exit, either use ctrl-c to kill the program from the terminal, or ctrl-d to send an EOF (end of file) character, which will cause getline to fail (which we catch inside the if statements).

Files

At some point, the day will come when you need to read or write a file. We can't rely on the disk giving us the data correctly, so there is a lot of error handling involved.

TL;DR: this is going to sting a bit.

No matter what we do with a file, we need to open it first. The only difference is the mode used to open the file.

ModeNameDescriptionIf file existsIf file does not exist
"r"readOpen a file for readingRead from startFail to open
"w"writeCreate a file for writingDestroy contentsCreate new
"a"appendAppend to a fileWrite to endCreate new
"r+"read extendedOpen a file for read/writeRead from startError
"w+"write extendedCreate a file for read/writeDestroy contentsCreate new
"a+"append extendedOpen a file for read/writeWrite to endCreate new

Additionally, "x" can be added, which forces the function to fail if the file exists. "t" can be added to force text mode (which is the default), where '\n', '\t', etc. are escaped into the relevant characters. "b" can be added to force binary mode where characters are not escaped.

To open a file, we need to call fopen, which has the following signature:

FILE* fopen(const char* filename, const char* mode);

This return a pointer to a file handle, and requires a filename and a mode from the table above.

Once we are done with a file handle, we can close it using fclose, which has the signautre:

int fclose(FILE* stream);

fclose returns 0 on success and EOF on fail.

A string can be written to a file using fputs or fprintf, which have the following signatures:

int fputs(const char* str, FILE* stream);
int fprintf(FILE* stream, const char* format, ...);

fprintf is identical to printf, but accepts a file handle instead of using stdout. The same '%' arguments can be used in any standard library function ending in printf.

To write a single character, use fputc:

int fputc(int ch, FILE* stream);

Reading a file is a bit more complicated. It is possible to read single characters using fgetc:

int fgetc(FILE* stream);

However, it is often easier to read the whole file at once. In this case, it is easier to create a function to read a whole file:

char* read_file(const char* filepath)
{
    FILE* file = fopen(filepath, "rt"); // open the file
    fseek(file, 0, SEEK_END); // go to the end of the file
    unsigned long length = ftell(file); // get the current position (aka the length of the file)
    char* data = (char*)malloc(length + 1); // allocate the string
    memset(data, 0, length + 1); // clear the string
    rewind(file); // go to start of file
    fread(data, 1, length, file); // read in the data
    fclose(file); // close the file
    return file; // return our string
}

This function can be called by supplying a file name. The string returned must be cleaned up using free once it is no longer needed.
