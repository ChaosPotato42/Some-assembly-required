Hello World

Let's start with the basics - a simple 'hello world' example. To do this we need to understand a few things. First, we need to understand where the program will start. In C and C++, this is at the main function. We will visit how this works later, but for now, all of the code that the program runs will live inside main. Create a file called main.cpp and follow the example.

main has the function signature:

int main()

This will also be explained later, but for now, let's get our hello world program working.

Our code must live inside a 'block', which is the area between two curly braces {}. This defines what is 'inside' a function. Any code that is outside of a block must be a definition or declaration (not code that runs) otherwise it will cause an error.

int main()
{
    // our code goes here
}

To be able to print a message, we need to include a file to give us access to printf. This is done using #include <stdio.h> or #include "stdio.h". The differences between these two methods will be explained later, but for now either will work perfectly fine.

#include <stdio.h>

...

Now we are ready to print our message using printf.

#include <stdio.h>

int main()
{   // the '\n' character prints a new line
    printf("Hello world\n");
}

This program will compile and run, but main expects an exit status. This is the exit status of the program once it terminates, and is used to tell if a program has failed. To do this we must 'return' a number from 0 to 255 as our exit status. 0 indicates the program was successful, anything else is an error code.

#include <stdio.h>

int main()
{
    printf("Hello world\n");

    return 0;
}

Our program is now complete - but we can't run it yet. It needs to be compiled. For a simple program like this, it is easy. Open a terminal/console in the directory containing main.cpp and type g++ main.cpp -o main to compile the program. It can now be run using ./main and you should see 'Hello world' appear in the terminal.

Note: if you have make installed, you can alternatively compile a single file project using make main.
